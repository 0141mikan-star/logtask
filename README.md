# ✅ logtask （勉強時間・タスク管理アプリ）

「勉強が続かない…」「タスク管理が味気ない…」そんな悩みを解決！ タスク消化や勉強時間を記録することで、経験値(XP)やコインが貯まる「ゲーミフィケーション」を取り入れたタスク管理アプリです。 貯めたコインでアプリのデザインを自分好みにカスタマイズしたり、ランキングで他のユーザーと競い合ったりしながら、楽しく継続することができます。

**🚀 デモアプリはこちら:** [https://logtask.streamlit.app/]

## ✨ 主な機能
🎮 楽しく続く「ゲーミフィケーション」
レベルアップシステム: 勉強したりタスクを完了するとXP（経験値）を獲得。レベルが上がると称号が変わります。

コイン＆ショップ: 頑張った分だけコインをGET！ショップで新しい「フォント」や「壁紙」を購入できます。

ログインボーナス: 毎日ログインするだけでコインがもらえます。

週間ランキング: その週の勉強時間を他のユーザーと競い合うことができます。

⏱️ 集中をサポートする「学習記録」
集中タイマー: 科目を選んでタイマーをスタート。終了時に自動で記録され、盛大に褒めてくれます（風船が飛びます🎉）。

手動記録: タイマーを使い忘れても、後から時間を入力可能。

データ分析: 過去の勉強時間や科目ごとの比率をグラフで可視化。頑張りが一目でわかります。

🎨 自由自在な「デザインカスタマイズ」
カラー変更: メイン文字色、サイドバーの文字色、強調カラー（アクセント）をカラーピッカーで自由に設定可能。

視認性調整: 背景の暗さやウィンドウの不透明度をスライダーで微調整し、自分が見やすい画面を作れます。

テーマ着せ替え: ショップで購入した「ドット絵風フォント」や「手書き風フォント」、様々な「壁紙」に着せ替え可能。

📝 基本もしっかり「タスク管理」
ToDoリスト: 期日設定、優先度設定が可能なタスク管理機能。

カレンダー連携: 登録したタスクや勉強記録をカレンダー形式で確認できます。

1日の目標設定: 自分で決めた目標時間を達成するとボーナスコインがもらえます。

🛠 使用技術
Frontend: Streamlit

Database: Supabase (PostgreSQL)

Language: Python

Visualization: Altair (グラフ描画), Streamlit Calendar

Authentication: Supabase Auth (Email/Password)

💻 ローカルでの実行方法
このリポジトリをクローンして実行する場合の設定です。

ライブラリのインストール

Bash
pip install -r requirements.txt
Supabaseの設定 .streamlit/secrets.toml ファイルを作成し、SupabaseのURLとAPIキーを設定してください。

Ini, TOML
[supabase]
url = "あなたのSupabaseプロジェクトURL"
key = "あなたのSupabase Anon Key"
アプリの起動

Bash
streamlit run app.py
🗄️ データベース構成 (Supabase SQL)
アプリを動作させるには、以下のテーブルが必要です。

users: ユーザー情報（XP, コイン, デザイン設定, カラー設定など）

tasks: タスク情報

study_logs: 勉強時間の記録

subjects: 科目マスタ
